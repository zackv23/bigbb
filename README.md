# BigBankBonus Website

Static HTML landing page for BigBankBonus.com

## Folder Structure

```
website/
├── index.html          # Main landing page
├── css/
│   └── style.css       # Styling for all pages
├── js/
│   └── main.js         # JavaScript functionality
├── assets/
│   └── images/         # Image files (add logos, icons, screenshots here)
└── README.md           # This file
```

## Files Overview

- **index.html** - Single landing page with:
  - Header with navigation
  - Hero section with call-to-action
  - Features showcase
  - How it works process
  - Contact form
  - Footer

- **css/style.css** - Responsive styling with:
  - Mobile-first design
  - Modern color scheme (blue theme)
  - Hover effects and transitions
  - Mobile and tablet breakpoints

- **js/main.js** - Lightweight JavaScript for:
  - Smooth scrolling navigation
  - Contact form handling
  - CTA button tracking

## Getting Started

1. Add your logo/images to `assets/images/`
2. Update the content in `index.html` with your actual information
3. Customize colors in `css/style.css` if needed
4. Deploy all files to your web hosting

## Uploading to BigBankBonus.com

All files in this directory are ready to upload to your web server:
- Upload the entire folder to your hosting provider
- Ensure the folder structure is preserved
- The `index.html` should be in the root directory of your domain

## Customization Tips

- **Logo**: Replace "BigBankBonus" text in header with an image
- **Colors**: Main blue color is `#0066cc` - change in CSS for different theme
- **Content**: Edit section headings and descriptions in `index.html`
- **Images**: Add images to `assets/images/` and reference them in HTML

## Browser Compatibility

- Modern browsers (Chrome, Firefox, Safari, Edge)
- Mobile responsive (works on phones and tablets)
- No framework dependencies - pure HTML, CSS, JS
